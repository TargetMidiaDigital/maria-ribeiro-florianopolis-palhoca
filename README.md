## Estrutura

```
apresentacao.html              ← home: seleção de unidade
index.html                     ← redireciona para apresentacao.html (GitHub Pages)
assets/logo-dark.png           ← logo Target (capa dos relatórios)
florianopolis/
  apresentacao.html            ← seleção de mês + gráfico de desempenho
  logo.jpg                     ← logo da unidade
  maio/apresentacao.html       ← relatório de Maio/2026
  maio/creativos/              ← vídeos dos melhores criativos
palhoca/
  apresentacao.html
  logo.jpg
  maio/apresentacao.html
  maio/creativos/
```
